📖 KETERANGAN

Atribut ( - ) = private / data yang disimpan
Method ( + ) = public / fungsi yang bisa dipanggil
Parameter = data yang diterima method
1 ─── * = Satu ke banyak (One to Many)
1 ─── 1 = Satu ke satu (One to One)
Class = Cetakan untuk membuat objek
